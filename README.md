Title: Breaking Bad Quotes using and API.

Description: SPA created using the create-react-app and https://breaking-bad-quotes.herokuapp.com to retrieve the list of quotes from the famous tv show.

Objective: Provide an interface where the user can click and obtain quotes from the tv show.
